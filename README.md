# Full Stack Experiments

Six deployable web development experiments, organized one folder per experiment.

## Experiments

4. `experiment-04` - E-commerce Catalog with nested schema concepts, variants, reviews, aggregation, indexes, and stock updates.
5. `experiment-05` - Express Middleware Implementation for logging, authentication, sequencing, and error handling.
6. `experiment-06` - MERN Integration using React-style components, Express API concepts, Axios flow, loading and error states.
7. `experiment-07` - Login Form with React state-management concepts, validation, loading, and auth feedback.
8. `experiment-08` - Dockerized React Application with multi-stage Dockerfile and Nginx production config.
9. `experiment-09` - Blog Platform with users, profiles, posts, comments, auth flow, and protected actions.

Each experiment has a browser demo that can be deployed independently on Netlify.
