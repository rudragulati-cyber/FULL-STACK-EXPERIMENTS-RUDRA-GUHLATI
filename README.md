# Full Stack Experiments

Six deployable web development experiments, organized one folder per experiment.

## Experiments

1. `experiment-01` - E-commerce Catalog with nested schema concepts, variants, reviews, aggregation, indexes, and stock updates.
2. `experiment-02` - Express Middleware Implementation for logging, authentication, sequencing, and error handling.
3. `experiment-03` - MERN Integration using React-style components, Express API concepts, Axios flow, loading and error states.
4. `experiment-04` - Login Form with React state-management concepts, validation, loading, and auth feedback.
5. `experiment-05` - Dockerized React Application with multi-stage Dockerfile and Nginx production config.
6. `experiment-06` - Blog Platform with users, profiles, posts, comments, auth flow, and protected actions.

Each experiment has a browser demo that can be deployed independently on Netlify.
