Experiment 05

Dockerize React Application

Multi-stage build, optimized production image, Nginx serving on port 8080, gzip, caching, and environment variables.

Build Stage

Node 18 installs dependencies and creates optimized static assets.

Runtime Stage

Nginx serves the compiled React app with production cache headers.

Expected Output

Production-ready Docker image under 100MB served on port 8080.

Dockerfile

FROM node:18-alpine AS build
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

FROM nginx:alpine
COPY nginx.conf /etc/nginx/conf.d/default.conf
COPY --from=build /app/dist /usr/share/nginx/html
EXPOSE 8080
CMD ["nginx", "-g", "daemon off;"]
